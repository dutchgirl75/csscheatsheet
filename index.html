Common CSS Properties

Color and Background Properties

  • Color- Sets the color of the text
  • Background-Color- Sets the background color of an element
  • Background-image- Sets one or more background color images for an element
  • Background-position- Sets the initial position of the background
  • Background-repeat- Sets if/how a background image will be repeated
  • Background-size- Specifies the size of the background images

Text Properties

  • Font-family- Specifies the font family for text
  • Font-size- Specifies the size of the font
  • Font-weight- Specifies the weight (boldness) of the font
  • Text align- Specufues the horizontal alignment of text
  • Text decoration- Sepecifies the decoration added to text (e.g. underline)
  • Line-height- Sets the line height

Box Model Properties

  • Width- Sets the width of an element
  • Height- Sets the height of an element
  • Margin- Sets the margin area on all four sides of an element
  • Border- Sets the border around an element
  • Box-sizing- Defines how the width and height of an element are calculated (including padding and borders or not)

Layout Properties

  • Display- Specifies how an element is diplayed (e.g. block, inline, flex)
  • Position- Specifies the type of positioning method used for an element (static, relative, absolute, fixed, sticky)
  • Top, Right, Bottom, Left- Specifies the position of a positioned element
  • Float- Specifies wether an element should float to the left, right, or not at all
  • Clear- Specifies what should happen with the element that is next to a floating element

Flexbox Properties

  • Flex- A shorthand property for the flex-grow, flex-shrink, and flex-basis properties
  • Flex-direction- Specifies the direction of the flexible items
  • Justify-content- Specifies the alignment between the items inisde a flexible container when the items do dont use all available space
  • Align-items- Specifies the alignment for items inside a flexible container

Grid Properties

  • Grid-template-columns- Specifies the size of the columns, and how many columns in a grid layout
  • Grid-template-rows- Specifies the size of the rows in a grid layout
  • Grid-column- A shorthand property for the grid-column-start and the grid-row-end properties
  • Grid-row- A shorthand property for the grid-row-start and the grid-row-end-properties

Animation Properties

  • Animation- A shorthand property for all the animation properties (animation-name, animation-duration, animation-timing-function, animation delay, animation-iteration-count, animation-fill-mode, animation-play-state)
  • Transition- A shorthand property for all the transition properties (transition-property, transition-duration, transition-timing-function, transition-delay)
